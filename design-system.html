Overview

새김 — 한 줄을 카드로, 발견하고 새기다

새김은 문장을 카드로 만들어 발견·새김·컬렉션으로 엮는 모바일 웹 서비스다. 한 줄 정의는 "한 줄을 카드로 만들어, 발견하고, 마음에 새겨 간직하는 곳." 깊이 있는 문장 큐레이션에 가벼운 작가 구독(인스타식)이 더해진다.

한 줄 정의

한 줄을 카드로 만들어, 발견하고, 마음에 새겨 간직하는 곳. 깊은 큐레이션 + 가벼운 작가 구독.

무게중심

깊은 큐레이션 플랫폼(문장·독서·카피 애호가 니치). 입구는 가벼운 발견 피드 → '가볍게 들어와 깊게 빠지는' 구조.

타겟

독서·문구 애호가, 카피라이터·기획자, 일상 문구를 기록·공유하려는 SNS 세대.

브랜드 '새김' · 하이브리드 톤

① 좋은 문장을 마음에 새기다(각인·소장) ② 글자를 새겨 남기다(출판 어원). 감성은 유지하되 비주얼·공유는 트렌디. 판각/도장(刻) 모티프는 제거.

핵심 루프

발견 · 포착 → 새김(서랍에 간직) → 재발견(피드 · 추천) → 곁에 두기 · 공유 ↻

문장을 만나고(발견) 만들어(포착) 서랍에 새기면, 추천·피드로 다시 떠오르고(재발견), 글벗을 곁에 두며 다시 새 문장으로 순환한다.

통합 데모 app.html

흩어져 있던 화면을 하나의 실행 가능한 앱으로 엮었다 — saegim/app.html(단일 파일 SPA, 바닐라 JS). 하단 5탭(홈·발견·＋포착·둘러보기·나)으로 전환되고, 카드·글을 누르면 발견 피드로 점프하며, 새김·좋아요·구독(글벗)이 실제로 동작한다(localStorage 저장). 아래 각 화면 섹션의 폰 데모는 모두 이 통합 앱을 해당 화면(#home · #discover · #search · #shelf · #me · #drawer)으로 띄운 것이다. ＋포착은 별도 saegim/editor.html로 연결된다.

Foundation

핵심 개념 모델

발행 단위는 늘 글, 원자 단위는 카드(장). 두 핵심 행동인 좋아요와 새김은 '보내는 것 vs 담는 것'으로 분리한다.

카드(장) ↔ 글 1:N

카드(장) = 원자 단위(한 종류, 긴 글은 카드 내 클립). 글 = 발행 단위(게시물)의 UI 명칭 = 카드(장)의 순서 묶음(1~N장). 1장도, N장도 한 '글'이다. UI 용어는 글/장, 내부 코드 식별자(COLLECTIONS 등)는 유지한다.

결(테마/무드)은 사용자가 직접 고르지 않는다

카테고리 강요를 지양한다. 비슷한 문장 묶기·추천·취향은 의미 임베딩(백엔드)이 담당한다(프로토타입은 클라이언트 similar()로 대체). 발행 시 입력은 보조 태그(사용자 자기 말, 0~3개, 선택)뿐이다.

좋아요 ↔ 새김 — 보내는 것 vs 담는 것

공개 공감

좋아요 ♡

남에게 보내는 가벼운 반응. 수치 공개, 저자 신호. 미리보기·상세에 ♡ 수 노출.

비공개 간직 · 브랜드 핵심

새김 (서랍에 넣기)

나만 보는 보관함(서랍)에 담아 다시 본다. 수치 비공개. 새김이 브랜드의 핵심 액션.

소셜 · 평등 계정

  • 인스타식 평등 계정 — 모든 계정이 동등하고 상호 구독한다. 구독 = 글벗(곁에 두기).
  • 인증 = 신분이 아닌 '상태' — 공식(서비스 공인)은 닉네임 옆 ✓ 채움 마크, 공인(본인/유산)은 가는 ✓.
  • 프로필 지표 = 글 수 + 글벗 수. 문장 수는 글과 혼동되어 비노출. 과열(경쟁 지표)은 절제.

카드 · 데이터 모델

카드(장)는 새김의 원자 단위이자 사용자의 창작물이다. 발행 단위는 늘 글(1~N장). 작성 구성(comp)을 통째로 보존해 작성=조회를 보장한다.

WYSIWYG 계약 — comp

에디터 comp{bg, dim, textColor, size, weight, align, font} ↔ 앱 composedInner/.cmp-text. 폰트 클래스 f-round/f-pen/f-black, 문구 박스는 콘텐츠 자동 너비(width:max-content; max-width:90%), 카드 내부 스크롤 없음. 이 계약을 바꾸면 app·editor 양쪽 파일 + 테스트를 동시 수정한다.

B2B 확장

출판사·작가 계정이 홍보 도서의 글귀를 카드로 발행한다. 신간 프로모션·광고·크리에이터 수익의 씨앗.

  • 진입점 — 에디터 '책 연결' 옵션, 글 상세 '책으로 보기'(현재 '준비 중')
  • 출판사·작가 계정도 평등 계정 모델 위에 얹힌다(인증=상태)
  • 콜드스타트 — '에디터 픽' 공식 큐레이션 + 작가·출판사 협업 계정
System

로드맵 · MVP

출시 형태는 웹(모바일 뷰) + 웹앱(WebView 래핑). 단일 컬럼·세로 스크롤·하단 5탭 + ＋포착. 데스크톱도 모바일 뷰 + 마우스/키보드 보강.

한 줄 요약

새김은 한 줄을 카드로 만들어 발견하고 마음에 새겨 간직하는 깊은 큐레이션 플랫폼이다. 카드(장)가 원자, 글이 발행 단위, 새김이 핵심 액션, 작가 구독(글벗)이 양념.
