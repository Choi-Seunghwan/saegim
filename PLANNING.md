# 새김 (SAEGIM) — 기획 노트

> 새김(SAEGIM) 독립 프로젝트 · 현재 **기획/디자인 단계** (개발 착수 전)
> 최종 갱신: 2026-06-28 · 본 문서 상단(1~13장)은 **현재 확정 상태 종합본**, 하단 **진행 로그**는 변경 이력(시간순).

---

## 1. 개요 · 한 줄 정의
- **새김(SAEGIM)** = **문장을 카드로 만들어 발견·새김·컬렉션으로 엮는** 모바일 웹 서비스.
- 한 줄: **"한 줄을 카드로 만들어, 발견하고, 마음에 새겨 간직하는 곳."** 깊이 있는 문장 큐레이션 + 가벼운 작가 구독(인스타식).
- 무게중심: **깊은 큐레이션 플랫폼**(문장·독서·카피 애호가 니치). 입구는 가벼운 발견 피드 → '가볍게 들어와 깊게 빠지는' 구조.
- 타겟: 독서·문구 애호가, 카피라이터·기획자, 일상 문구를 기록·공유하려는 SNS 세대.

## 2. 브랜드 · 정체성
- **브랜드명 '새김'** — ① 좋은 문장을 *마음에 새기다*(각인·소장) ② 글자를 *새겨 남기다*(출판 어원). 동사 '새기다'.
- **하이브리드 톤**: 새김의 감성(마음에 새겨 간직)은 유지하되, 비주얼·공유는 트렌디(인스타 감성카드). **판각/도장(刻) 모티프는 제거**(발행 시 잔존 모션도 삭제). 비주얼 방향 **'C(소프트 감성)' 확정**.
- 캐릭터 호칭(시인·큐레이터 등)은 정체성 표기에서 내림 — **닉네임 + 인증 상태 + 한줄 소개글**로 표현.

## 3. 비주얼 시스템 (C · 모노톤)
- **원칙**: 색은 **'글 카드'(그라데이션)에만**, UI 크롬은 무채색. 외곽선 최소화·버튼은 심리스(채움/보더 지양, 주 액션만 톤 채움).
- **컬러 토큰**: 배경 뉴트럴 미스트 `#F6F5F6` · 잉크(먹보라/차콜) `#38323F` · **포인트=잉크 차콜 `#353039` 모노톤** · 보조 무채색. (분홍 제거 — 첫 적용이 과하게 분홍→모노톤 정정.)
- **카드 배경 프리셋**: 새벽 · 노을 · 안개 · 살구 · 라벤더 · 밤 그라데이션 + 단색(밝은/어두운) + 패턴 + 이미지. **에디터 기본=안개(중성)**. 이미지 배경은 딤(어둡게) 레이어로 가독 확보.
- **폰트**: 카드 문장 **Gowun Batang**(소프트 명조) · 워드마크 **Gowun Dodum**(잉크 밑줄) · UI **Pretendard**.
- 대안 D(다크 심야 감성)는 `saegim/brand-d.html`에 보존.

## 4. 핵심 개념 모델
- **카드(장)** = 원자 단위(한 종류, 긴 글은 카드 내 스크롤). **글** = 발행 단위(게시물)의 UI 명칭 = 카드(장)의 순서 묶음(1~N장).
- **카드(장) ↔ 글 = 1:N**. 1장도, N장도 한 '글'. UI 용어는 **글/장**, 내부 코드 식별자(COLLECTIONS 등)는 유지.
- **결(테마/무드)은 사용자가 직접 안 고름**(카테고리 강요 지양). 비슷한 문장 묶기·추천·취향은 **의미 임베딩(백엔드)** 담당(프로토타입은 클라 similar). 발행 시 입력은 보조 **태그**(사용자 자기 말, 0~3, 선택)뿐.
- **좋아요 ↔ 새김 분리(둘 다 유지)** — *"보내는 것 vs 담는 것"*:
  - **좋아요(♡) = 공개 공감** — 남에게 보내는 가벼운 반응, 수치 공개, 저자 신호.
  - **새김(서랍에 넣기) = 비공개 간직** — 나만 보는 보관함(서랍)에 담아 다시 봄, 수치 비공개. **브랜드 핵심 액션.**
- **소셜 = 인스타식 평등 계정**: 모든 계정 동등·상호 구독. **인증=신분 아닌 '상태'**(공식=닉네임 옆 채움 마크, 공인=가는 ✓). 구독 = **글벗**(곁에 두기). 과열 요소(경쟁 지표) 절제.
- **프로필 지표 = 글 수 + 글벗 수**(구독자→'글벗' 순화). **문장 수는 비노출**(글과 혼동 방지).

## 5. 정보구조 — 5탭 + 검색
- 하단 **5탭(라벨 없는 아이콘만)**: 홈 · 발견 · ＋포착 · 둘러보기 · 나(=프로필 사진). 상단 우측 **검색** 아이콘.
- **홈** = 에디토리얼(서비스 주도). **발견** = 글 피드(몰입 탐색). **＋포착** = 카드 에디터(별도 페이지, 서비스의 심장). **둘러보기**(구 '서가') = 글 갤러리(사용자 주도). **나** = 통합 프로필.
- 전체 페이지(설정·프로필 편집)는 하단 탭·FAB 숨김.

## 6. 화면별 상세
### 홈 (에디토리얼)
- 상단 **캐러셀 배너**: '오늘의 문장' + 추천 글 몇 개를 가로 스와이프(점 인디케이터·자동 전환·드래그, 탭→상세/피드).
- **오늘 닿은 글** 가로 레일(미리보기 카드, 데스크톱 마우스 드래그 스크롤).
- **추천 글벗** 가로 레일: 아바타·닉네임(+공식 마크)·**한줄 소개글**·**글 수·글벗 수**·구독 버튼.
- 제거: 테마칩(테마로 둘러보기), '당신을 위한 문장'(미합의 기능).

### 발견 (글 피드)
- **2D 뷰어**: 세로=글 이동, 가로=장(카드) 이동. 모바일 스와이프 + **PC 마우스 휠/키보드 방향키/드래그**(긴 본문은 휠로 스크롤하다 끝에서 글 전환).
- 크롬: 상단 **작가 바**(심리스 프로스트, 아바타·이름·상태·구독), 상단 중앙 **제목 라벨**(현재 문장과 같으면 숨김), 하단 중앙 **장 점 인디케이터**, 우측 **액션 레일**(새김·좋아요·댓글, 투명 아이콘).
- 우측 상단 **⋯ → 하단 정보 패널**: 출처(『책』·저자), 태그, 공유하기 / 링크 복사(준비) / **책으로 보기(B2B·준비)** / 신고(준비).

### 둘러보기 (글 갤러리)
- 부제 '엮어 둔 글 모음'. 정렬 **인기/최신**(무드 필터는 MVP 보류 — 백엔드 큐레이션 전제). 메이슨리 카드 그리드.
- 카드 진입 → 발견 피드로 점프.

### 검색
- 계정·글·문장 통합 결과 + 빈 입력 시 최근 검색·인기. 계정 메타=한줄 소개글·글벗 수. (무드 찾기 제거.)

### 나 / 프로필 (나 = 타인 동일 레이아웃)
- 상단바: 아이콘만(나=옵션, 타인=뒤로). **헤더 닉네임 중복 제거.**
- 헤더: 아바타(내 사진 반영) · 이름+**공식 마크** · **한줄 소개글** · **글 N개 · 글벗 M** · 우측 **컴팩트 액션 버튼**(나=프로필 편집 / 타인=곁에 두기).
- 그 아래 **소개글(인용)** — 한줄 소개글과 다를 때만(따옴표 아이콘 제거).
- **글 그리드**: 길이 가변 카드, 우측 상단 **'N장' 배지**(미리보기 전반 통일), 작성자 미노출(이미 특정됨).
- **설정 전체 페이지**: 계정(프로필 편집·계정 정보) / 활동(내 서랍·구독) / 알림 / 정보 / 로그아웃·초기화. (내 서랍은 설정에 잔존, 프로필에선 제거.)
- **프로필 편집 전체 페이지**: 프로필 사진(업로드·미리보기) · 닉네임 · 한줄 소개글 · 프로필 소개글(글자수 카운터) → `state.profile` 저장·반영.

### 서랍 (새김 보관함)
- '내가 간직한 글 · 나만 봐요'(비공개). 뒤적이기(덱)·목록 모드, 검색·정렬. 새김 시 여기로 모임. 진입은 설정의 '내 서랍'.

### 글 상세 댓글
- 발견 액션바 **댓글 버튼+개수** → 댓글 시트(목록 + 작성칸). 프로토타입은 `state.comments` 키별 저장 + 샘플 시드.

### 진입 게이트(인증)
- 미로그인 진입 시 **로그인/회원가입**(이메일·비번, 카카오/Apple 자리) + 게스트('로그인 없이 둘러보기'). 프로토타입은 `state.loggedIn` 플래그. (실 OAuth·검증은 백엔드 연동 시.)

## 7. 카드 에디터 (＋포착) — 작성 = 조회 동일(WYSIWYG)
- **풀블리드 캔버스**: 카드가 화면을 꽉 채워 조회(발견/뷰어)와 동일 뷰. 발행 시 구성(comp)을 통째로 전달해 어디서나 같게 렌더.
- **인라인 편집**: 캔버스/문구 탭 → 키보드 + 폰트(명조/고딕)·크기·정렬·굵기·**글자색(프리셋 + 직접 색상 픽커)**. 문구·출처는 **드래그로 위치 이동**, 탭하면 재편집.
- **우측 상단 도구(바텀시트) 4종**: 제목 · 배경 · 출처 · 태그. (꾸밈/스티커는 MVP에서 제거.)
  - **배경**: **직접 색상 픽커(최상단)** + 프리셋(밝은/어두운 단색·그라데이션·패턴·이미지) + **딤(어둡게) 슬라이더**.
  - **출처**: **저자 · 책**(연도 필드 제거). 책 연결은 B2B 확장 자리.
  - **제목**: 선택 입력(비우면 첫 문장 자동). 입력 시 카드 상단 제목 라벨 표시.
  - **태그**: 0~3개 자유 태그(검색·유사 추천 힌트).
- **발행**: 미리보기 모달 없이 **발행 확인 팝업** 후 핸드오프 큐 적재 → 앱이 수신해 발견·둘러보기·프로필에 반영. **카드 삭제 시도 확인 팝업**(되돌릴 수 없음 안내).
- 톤: 입력칸·링크 보더 완화(무채색 채움+포커스 은은한 링), 버튼 심리스 — 상세보기와 통일.

## 8. 소셜 · 계정 · 인증
- 모든 계정 동등(일반·작가·출판사 동일 모델), 누구나 상호 구독(**글벗**). 프로필은 나=타인 동일 화면(나만 옵션 메뉴).
- **인증 = 상태**: 공식(서비스 공인) = 닉네임 옆 **채움 마크**(어떤 배경에서도 보이게 even-odd 체크), 공인(본인/유산) = **가는 ✓**. 캐릭터 호칭 미표기.
- 표시: 닉네임 옆 마크 + 보조 줄 **한줄 소개글**('공식 계정' 칩은 제거, 마크로 대체).

## 9. 데이터 모델
### 개념(백엔드 방향)
- **카드(장)**: 본문 · 배경/스타일/위치(comp) · 출처 · 소속 글 id · 순서 · 임베딩.
- **글(Collection)**: 제목 · 작성자 · 카드 목록(순서) · 커버 · 공개여부 · 생성유형(창작/큐레이션) — 카드와 **1:N**.
- **계정**: 프로필(사진·닉네임·한줄 소개글·소개글) · 상호 구독 그래프 · 서랍(새김) · 발행 글 · 인증 상태.
- **출처**: 유형(책/웹/직접/출판사) · 저자 · 책 · URL.
- 임베딩 → 취향 피드 · 유사 추천 · 자동 묶음. 관계: 발행=글(≥1장) · 발견=글/장 · 서랍=새김(비공개) · 좋아요=공개 · 구독=계정↔계정.

### 프로토타입(현재 구현, localStorage·단일 HTML)
- 키: `saegim_v1`(앱) · `saegim_editor_v2`(에디터) · `saegim_pub_queue`(발행 핸드오프).
- 앱 `state`: carved(새김) · liked(좋아요) · subs(글벗) · published(발행 글) · comments · profile{name,photo,bio,tagline} · loggedIn · taste(휴면).
- 에디터→앱: 발행 시 글(제목·장들·comp·테마·태그)을 큐에 적재 → 앱이 진입/복귀 시 흡수.

## 10. 콘텐츠 · 시드 · 저작권
- 안전 시드: 공유저작물(작고 후 충분 경과) + 고전·명언 + 자체 창작/출판. 현대물은 저자 라이선스만.
- 소스 예: 위키문헌 · 공유마당 · 한국고전종합DB · Project Gutenberg.
- 콜드스타트: '에디터 픽' 공식 큐레이션 + 작가·출판사 협업 계정. 초기엔 홈 큐레이션·추천 중심.

## 11. 기술 · 플랫폼 · 작업 규칙
- **출시 형태**: 웹(모바일 뷰) + 웹앱(WebView 래핑). 단일 컬럼·세로 스크롤·하단 5탭 + 플로팅 포착. 데스크톱도 모바일 뷰 + 마우스/키보드 보강.
- **데모 원칙**: 의존성 없는 **단일 HTML**(바닐라 JS, Pretendard + Gowun Batang/Dodum CDN, C 모노톤 토큰). 변경 후 jsdom으로 동작 검증.
- **현재 검증 상태**: jsdom **app 113 / editor 63** 통과(작성 시점).

## 12. 비즈니스 확장 (B2B)
- 출판사·작가 계정 → 홍보 도서 글귀를 카드로 발행. 에디터 '책 연결' 옵션·글 상세 '책으로 보기'가 진입점(현재 '준비 중').
- 신간 프로모션·광고·크리에이터 수익의 씨앗.

## 13. MVP 범위 · 백로그
