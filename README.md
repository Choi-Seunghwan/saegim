# 새김 (SAEGIM)

> 문장을 카드로 만들어 **발견하고 · 새기고 · 엮는** 모바일 웹 서비스
> 현재 **기획/디자인 단계** (개발 착수 전)

## 한 줄 정의
좋은 문장을 카드로 만들어 발견하고 마음에 새기는 곳. 깊이 있는 문장 큐레이션 + 인스타식 평등 계정 소셜.

## 핵심 개념
- **카드** = 원자 단위(문장 + 스타일 + 출처). 한 종류이며, 긴 글은 카드 안에서 스크롤한다.
- **컬렉션** = 카드들의 순서 있는 묶음. **발행 단위**(카드 1장도 1장짜리 컬렉션). 카드 ↔ 컬렉션은 **1:N**.
- **5탭**: 홈(에디토리얼) · 발견(릴스) · ＋포착(에디터) · 서가(컬렉션) · 나 + 검색
- **소셜**: 모든 계정이 동등(일반·작가·출판사), 누구나 서로 구독. 인증 배지(✓)는 '상태' 표시.

## 폴더 구조
- `PLANNING.md` — 기획 노트(결정·진행 로그). **여기부터 읽으면 전체 맥락이 잡힌다.**
- `design-system.html` — 기획·디자인 시스템 문서(화면별 데모를 통합 앱 딥링크로 임베드)
- `saegim/` — 데모
  - `app.html` — **통합 앱 (메인 데모)**. 5탭 + 컬렉션 2D 뷰어 + 검색, 단일 파일 SPA
  - `editor.html` — 카드 에디터(포착)
  - `collection.html` — 컬렉션 2D 뷰어(단독 시연)
  - `home.html` · `shelf.html` · `curator.html` · `index.html` — 화면 단독 원본
- `concepts/` · `mockups/` · `mvp-mockups.html` · `gallery.html` — 초기 UI 탐색

## 데모 보는 법
브라우저에서 `saegim/app.html`(통합 앱)을 열거나, `design-system.html`(기획서 + 화면별 데모)을 연다.

## 디자인
따뜻한 종이 + 모던 — 출판의 물성(종이·잉크·각인)에 카드·릴스 문법을 얹는다. 본문 Pretendard, 강조에 세리프.

## 상태 / 저작권
기획·디자인 프로토타입. 모든 문장은 공유저작물 또는 자체 창작 시드. 실제 런칭 전 라이선스 검토 필요.
